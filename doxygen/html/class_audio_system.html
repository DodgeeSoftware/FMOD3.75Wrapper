Detailed Description

The AudioSystem class is the core of the Sound Wrapper The AudioSystem class intialises and shutdowns the audio system along with setting important things like volume, balance, allowing you to mute audio, get important information about the driver, record audio, play cds and add DSP (Digital Signal Processing) to the Audio Pipeline (for effects like echo)
