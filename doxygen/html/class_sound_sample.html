Detailed Description

SoundSample is a wrapper around and FSOUNDSample* The Sound Sample class is a wrapper around an FSOUND_SAMPLE object. This class is used because the FSOUND_SAMPLE object is actually a predeclaration and using is as anything other than a pointer will create compile errors. So it becomes impossible to bind the FSOUND_SAMPLE to any scripting language because the object isn't defined.
